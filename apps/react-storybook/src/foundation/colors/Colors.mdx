# Colors

Color is a **functional** and **emotional** tool. Our color system communicates brand personality, sets a tone, attracts attention, and indicates importance. In other words, it allows the creation of a web interface more usable and helps us consistent experiences across products.

Our color primitives are defined as design tokens and available as CSS custom properties in `:root`.

The primary brand color palette used across the design system.

**Note**: All color tokens are CSS custom properties defined in `:root` and can be used directly with `var()`.

> **_Example_**: Using color tokens
>
> ```css
> .my-component {
> 	color: var(--color-brand-500);
> 	background-color: var(--color-gray-800);
> 	border-color: var(--color-red-500);
> }
> ```
