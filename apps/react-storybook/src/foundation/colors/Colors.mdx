# Colors

Color is a **functional** and **emotional** tool. Our color system communicates brand personality, sets a tone, attracts attention, and indicates importance. In other words, it allows the creation of a web interface more usable and helps us consistent experiences across products.

## Main Colors

Every main color can have these _variations_ _(variations can only be used to
complement the base color)_:

-   **base**: main color
-   **contrast**: color that will provide readable text against the base color
-   **dark**: variation that should be a darker version of the base color(used in the _hover_ state of _Button_, for example)
-   **light**: variation that should be a lighter version of the base color(used in the _disable_ state of _Button_, for example)

## Neutral Colors

The _Neutral Colors_ consisted colors that **inform, create visual balance and harmony**. These colors are used to provide variations in text and background colors of components.
**Note**: Should be _Text Color_ stepping towards _Background Color_.

**Note**: All colors can be replaced by setting the respective _CSS Variable_

> **_Example_**: Setting the _Primary Color_

```css
--kv-primary: #005cc7;
--kv-primary-rgb: 0, 92, 199;
--kv-primary-contrast: #fff;
--kv-primary-contrast-rgb: 255, 255, 255;
--kv-primary-dark: #0051af;
--kv-primary-light: #ccdef4;
```
