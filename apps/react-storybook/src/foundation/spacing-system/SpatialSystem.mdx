# Spatial System

_Space is everywhere. The organization of space is key to every great design, providing **consistent rhythm** to all our products._

Our spatial system is built on design tokens generated by Style Dictionary. These tokens provide consistent spacing, sizing, radii, and border widths across all components.

## Spacing

Spacing tokens are used for margins, paddings, and gaps between elements.

<div className={styles.SpacingContainer}>
	{SPACINGS.map(({ token, value }) => (
		<div key={token} className={styles.SpacingWrapper}>
			<div className={styles.BoxWrapper}>
				<div
					className={styles.Box}
					style={{ width: value, height: value }}
				></div>
			</div>
			<div className={styles.Description}>
				<span className={styles.Code}>{token}</span>
				<span className={styles.Value}>{value}</span>
			</div>
		</div>
	))}
</div>

## Sizes

Size tokens are used for component dimensions like heights and widths.

<div className={styles.SpacingContainer}>
	{SIZES.map(({ token, value }) => (
		<div key={token} className={styles.SpacingWrapper}>
			<div className={styles.BoxWrapper}>
				<div
					className={styles.Box}
					style={{ width: value, height: value }}
				></div>
			</div>
			<div className={styles.Description}>
				<span className={styles.Code}>{token}</span>
				<span className={styles.Value}>{value}</span>
			</div>
		</div>
	))}
</div>

## Radii

Radius tokens define the border-radius for components.

<div className={styles.SpacingContainer}>
	{RADII.filter(({ token }) => token !== "--radius-full").map(({ token, value }) => (
		<div key={token} className={styles.SpacingWrapper}>
			<div className={styles.BoxWrapper}>
				<div
					className={styles.RadiusBox}
					style={{ borderRadius: value }}
				></div>
			</div>
			<div className={styles.Description}>
				<span className={styles.Code}>{token}</span>
				<span className={styles.Value}>{value}</span>
			</div>
		</div>
	))}
	<div className={styles.SpacingWrapper}>
		<div className={styles.BoxWrapper}>
			<div
				className={styles.RadiusBox}
				style={{ borderRadius: "999px" }}
			></div>
		</div>
		<div className={styles.Description}>
			<span className={styles.Code}>--radius-full</span>
			<span className={styles.Value}>999px</span>
		</div>
	</div>
</div>

## Border Widths

Border width tokens define the thickness of borders.

<div className={styles.SpacingContainer}>
	{BORDER_WIDTHS.map(({ token, value }) => (
		<div key={token} className={styles.SpacingWrapper}>
			<div className={styles.BoxWrapper}>
				<div
					className={styles.BorderBox}
					style={{ borderWidth: value }}
				></div>
			</div>
			<div className={styles.Description}>
				<span className={styles.Code}>{token}</span>
				<span className={styles.Value}>{value}</span>
			</div>
		</div>
	))}
</div>

**Note**: All tokens are CSS custom properties defined in `:root` and can be used directly with `var()`.

> **_Example_**: Using spacing tokens
>
> ```css
> .my-component {
> 	padding: var(--spacing-md);
> 	margin-bottom: var(--spacing-2xl);
> 	gap: var(--spacing-xs);
> 	border-radius: var(--radius-md);
> 	border: var(--border-width-default) solid;
> }
> ```
